Galeb Manager WebUI
===========================

Galeb Manager WebUI is a pretty interface to use [Galeb Manager](https://github.com/galeb/galeb-manager).<br/>


Building
-----

+ Install dependencies with [Bower](https://github.com/bower/bower).

>
```bash
$ bower install
```
